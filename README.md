## 🍄 Landing Page - Super Mario Brothers

## 📌 Sobre o Projeto

Esta landing page foi desenvolvida como uma homenagem inspirada no universo do filme **Super Mario Bros.**, trazendo elementos visuais que remetem à aventura, diversão e nostalgia da franquia.

O projeto busca recriar a identidade vibrante do Reino dos Cogumelos por meio de um layout moderno, destacando personagens, cenários e cores inspirados no longa-metragem, proporcionando uma experiência imersiva para os fãs.

## 🚀 Funcionalidades

* Layout inspirado no universo Super Mario Bros.
* Destaque para personagens e elementos icônicos da franquia
* Estrutura organizada e de fácil navegação
* Efeitos visuais modernos
* Interface intuitiva e atrativa

## 🎨 Design e Experiência

O design foi desenvolvido utilizando cores vibrantes e elementos que remetem ao filme e aos jogos da franquia. A proposta é transmitir a energia das aventuras de Mario, Luigi e seus amigos, criando uma experiência divertida e envolvente para os visitantes.

## 🔮 Tecnologias Utilizadas

- HTML5 🟧
- CSS3 🟦
- JAVASCRIPT 🟨

## 📱 Responsividade

A página foi desenvolvida para se adaptar a diferentes dispositivos, garantindo uma ótima experiência em computadores, tablets e smartphones.

## 💡 Objetivo

O objetivo deste projeto é demonstrar habilidades em desenvolvimento front-end, recriando uma landing page temática inspirada no universo Super Mario Bros. e aplicando conceitos de design responsivo, organização visual e experiência do usuário.

## ⚠️ Aviso

Este projeto possui finalidade exclusivamente educacional e de portfólio. Todos os personagens, nomes e elementos relacionados à franquia Super Mario pertencem à Nintendo e seus respectivos proprietários.
